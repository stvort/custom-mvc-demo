Астролябии

Астролябия — это древний инструмент, использовавшийся астрономами и навигаторами для измерения углов между небесными телами и горизонтом.

Она состоит из плоской диска с градуировкой и подвижной стрелки, называемой альтезой. Астролябии использовались для определения времени, широты и долготы, а также для навигации в море.

Этот инструмент был особенно популярен в средние века и стал основой для многих других астрономических приборов.

Астролябия позволяет пользователю проводить наблюдения за звездами и планетами, что делает её важным инструментом в астрономии и навигации.



Секстанты

Секстант — это навигационный инструмент, используемый для измерения углов между небесными телами и горизонтом.

Он был разработан в 18 веке и стал важным инструментом для мореплавателей. Секстант позволяет точно определять широту и долготу, что делает его незаменимым для навигации на море.

Устройство состоит из полукруга с градуировкой и подвижной линейкой, которая позволяет измерять углы. Секстанты использовались для определения местоположения кораблей и для астрономических наблюдений.

Этот инструмент значительно улучшил безопасность мореплавания и открыл новые горизонты для исследователей.
